#React-приложение Todo list

Приложение, позволяющее добавлять, удалять, фильтровать задачи. При добавлении / удалении / фильтрации есть анимация. 

# В проекте используются
- react
- typescript
- tailwindcss
- axios
- classnames
- mobx
- mobx-react-lite
- react-confetti
- react-transition-group
- typescript
